Introduction

Installation

Install database PostgreSQL.

Install Haskell compiler GHC and package installer cabal-install.

Environment variables

Variable Type Default Description
PORT number 8000 Socket port to listen on
DATABASE_URL connection string PostgreSQL database to use
LOG True / False False Display log messages

Initialize database

psql $DATABASE_URL < sql/setup.sql

Build

cabal v2-build

Execution

cabal v2-run

Other Commands

Help text
cabal v2-run help
Database migration
cabal v2-run migrate
Add user
cabal v2-run adduser {username} {password} {role}
